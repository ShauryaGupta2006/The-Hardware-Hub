# The Hardware Hub 🛠️🛒

The Hardware Hub is a **full-fledged website** built using **Node.js, Express.js, MongoDB, Mongoose, EJS**, and static assets.  
It is designed for a hardware store system where **Users, Owners, and Products** can be managed in an organized way.

---

## 🚀 Key Features

### 👤 User Module
- User Registration
- User Login (Authentication)
- User Profile / Dashboard
- Browse Products

### 🧑‍💼 Owner Module
- Owner Registration / Login
- Owner Dashboard
- Manage Products (Add / Update / Delete)

### 📦 Product Module
- Add Products
- View Product Listing
- Product Details Page
- Update Product Info
- Delete Product

### 🌐 Website Features
- Dynamic Pages using EJS
- Static Assets (CSS, JS, Images) via Public Folder
- MongoDB Database for storing all data
- Organized folder structure with routes + models
